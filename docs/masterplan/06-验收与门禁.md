# 06 — 验收清单与门禁

---

## A. UX（Grok 级 · 禁止退化）

### 冷启动
- [ ] 10 分钟：装→key→第一条成功回复  
- [ ] 10 分钟内完成一次文件编辑  
- [ ] doctor 说人话  

### TUI
- [ ] 流式流畅、可中断、可滚动  
- [ ] 审批清晰、模式可见、模型可见  
- [ ] CJK 不明显错位、大粘贴不炸  
- [ ] slash help 可用  

### 生产力回路
- [ ] read/edit/bash/grep 可靠  
- [ ] 多 tool 单轮  
- [ ] session resume  
- [ ] plan 只读硬闸  

### DeepSeek
- [ ] 默认或一键 DeepSeek  
- [ ] cache 可见  
- [ ] 本地兼容 endpoint  

### 安全
- [ ] smoke-security 绿  
- [ ] sandbox 推荐文档化  
- [ ] 遥测默认关  

### 差异
- [ ] verify 自修可演示  
- [ ] storm 可演示  
- [ ] 垂直 ≤3 步

## B. 阶段门禁（摘要）

| 阶段 | 必须 |
|------|------|
| M0 | 编译、DeepSeek tool、2 日自用 |
| M1 | smoke-security |
| M2 | cache、假完成、storm、5 日自用 |
| M3 | CC_PARITY ≥80% |
| M4 | editverify、垂直、eval≥20 |
| M5 | UX A–F、10 分钟新手 |
| M6 | 15 日自用、法律、v1.0 |

## C. 任务级 DoD（每个 PR）

- [ ] 用户可见行为或安全默认变更  
- [ ] 测试或 scripts 可复现  
- [ ] 不打开遥测、不恢复强制 xAI 登录  
- [ ] 真实摸过一次  

---

## D. 玩具判定（出现即不合格）

- README 有、主路径没有  
- 只 mock、无真实 DeepSeek 多轮  
- 双 TUI  
- 自己不用却发版  
- 安全仅文档
