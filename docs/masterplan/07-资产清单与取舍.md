## 回炉原则

若 `agent/` 被改乱：

```bash
rm -rf ~/code/lumen/agent

# 再 cherry-pick 自有 commits（安全/品牌/配置）
```

**自有逻辑尽量在独立文件 + hooks + config**，方便回炉后重放。
