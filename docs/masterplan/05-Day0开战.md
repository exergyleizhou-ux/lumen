# 05 — Day 0 开战（复制执行）

> 目标：当天结束 = monorepo 在盘上 + check 开始跑 + 阻塞点写清。

```bash
# 0. 环境
export PATH="$HOME/.cargo/bin:$HOME/.local/bin:$PATH"
rustc --version
xcode-select -p
# 磁盘建议空闲 ≥ 20GB

# 1. 建仓
mkdir -p ~/code/lumen
cd ~/code/lumen
git init

rsync -a --exclude target --exclude .git \
  ~/Desktop/grok-build-main/ ./agent/

mkdir -p packs policy skills evals scripts journal docs/masterplan
cp -R ~/Desktop/Lumen\ Masterplan/* docs/masterplan/ 2>/dev/null || true

cat > NOTICE << 'EOF'
Lumen agent runtime is derived from SpaceXAI Grok Build
(https://github.com/xai-org/grok-build), Apache License 2.0.
See agent/LICENSE and agent/THIRD-PARTY-NOTICES.
Vertical packs and modifications: Copyright Lumen authors.
EOF

cat > agent/UPSTREAM.md << 'EOF'
Source: xai-org/grok-build
Import: ~/Desktop/grok-build-main
Date: 2026-07-16
Policy: PINNED snapshot. Security-only cherry-picks. No auto-merge upstream.
EOF

cat > LEGAL.md << 'EOF'
See docs/masterplan and NOTICE.
Do not claim official xAI/SpaceXAI or Anthropic affiliation.
Telemetry must default off.
EOF

git add -A
git commit -m "chore: import grok-build as agent foundation (pinned)"

# 2. 编译（可能很长）
cd ~/code/lumen/agent
cargo check -p xai-grok-pager-bin 2>&1 | tee /tmp/lumen-check.log
# 通过后：
# CARGO_BUILD_JOBS=2 cargo build -p xai-grok-pager-bin --release 2>&1 | tee /tmp/lumen-build.log

# 3. 冒烟（二进制名以产物为准）
ls -la target/release/xai-grok-pager target/release/lumen 2>/dev/null
./target/release/xai-grok-pager --help 2>&1 | head -40
```

| 症状 | 动作 |
|------|------|
| protoc | 安装 protobuf 或修好 `bin/protoc` |
| 内存爆 | `CARGO_BUILD_JOBS=1` |
| 单 crate 错 | 只 `cargo check -p <crate>` 定位 |
