# 08 — M2 循环纪律（实现对照）

## 模块

| 能力 | 代码 | 接线 |
|------|------|------|
| StormBreaker | `lumen-discipline::StormBreaker` | 单测可演示；会话资源接入后续增强 |
| RepeatSuccessGuard | `lumen-discipline::RepeatSuccessGuard` | 同上 |
| Delivery turn-end | `on_turn_end` + `DELIVERY_REMINDER` | 纯逻辑；turn 边界注入待 shell 增强 |
| Goal incomplete SoftOnce | `gate_goal_complete` | **`update_goal` 工具入口已接线** |
| Cache 行格式 | `format_cache_line` | 状态栏/session-info 可消费 |
| Presets | `default_models.json` | chat / reasoner / local-openai |

## 假完成拦截（可演示）

1. 会话内 `todo_write` 留下 `pending`/`in_progress`
2. 模型调用 `update_goal(completed: true)`
3. 工具返回 error code `incomplete_todos` + detail
4. 再调用一次 `completed: true` → SoftOnce 放行（第二次）
