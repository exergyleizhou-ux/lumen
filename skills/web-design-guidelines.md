# Web Design Guidelines
Design principles for web interfaces:

1. **Visual hierarchy**: Size, color, spacing to guide attention. Important things first.
2. **Consistent spacing**: Use a spacing scale (4px, 8px, 16px, 24px, 32px, 48px, 64px).
3. **Typography**: Max 2 font families. Line height 1.5 for body, 1.2 for headings. Max line width 65ch.
4. **Color**: 60% neutral, 30% primary, 10% accent. Ensure WCAG AA contrast (4.5:1 for text).
5. **Accessibility**: Semantic HTML, focus indicators, alt text, ARIA labels where needed.
6. **Responsive**: Mobile-first. Break at 640px, 768px, 1024px, 1280px.
7. **Loading states**: Skeleton screens or spinners. Never flash of no content.
8. **Empty states**: Guide the user when there's nothing to show.
9. **Error states**: Clear error messages near the problem. Validate early.
